Mobile (Frontend)

- React Native via Expo
- React Navigation (stack + tab navigation + drawer navigation)
- Redux Toolkit (state management for products, orders, reviews)
- SQLite via expo-sqlite (cart persistence)
- Expo SecureStore (JWT token storage)
- Expo Camera / Image Picker (photo uploads)
- Expo Notifications (push notifications)


Backend

- Node.js + Express.js
- MongoDB + Mongoose (main database)
- JWT (authentication)
- Multer (file handling middleware)
- Expo Push Notifications SDK (server-side push sender)


Auth & Social Login

- Firebase Authentication (Google + Facebook OAuth)
- Firebase Admin SDK (server-side token verification)


Media

- Cloudinary (image storage + delivery for products and avatars)


Dev Tools

- ESLint + Prettier
- Nodemon
- dotenv
